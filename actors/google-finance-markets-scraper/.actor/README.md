# Google Finance Markets Scraper

Scrape Google Finance market movers, trend rows, major indices, currencies, crypto, futures, and finance news. The actor writes one dataset item per market row or news result and stores the raw Scrappa response in `OUTPUT`.

Use the default input for gainers, or set `trend` to `losers`, `most-active`, `indexes`, `climate-leaders`, `cryptocurrencies`, or `currencies`.
